## Configuration Management

- **Ansible**: [ansible/ansible](https://github.com/ansible/ansible)
  - Simple, agentless IT automation platform.
- **Chef**: [chef/chef](https://github.com/chef/chef)
  - Infrastructure automation for managing and configuring servers.
- **Puppet**: [puppetlabs/puppet](https://github.com/puppetlabs/puppet)
  - Server automation framework and application.
- **SaltStack**: [saltstack/salt](https://github.com/saltstack/salt)
  - Infrastructure automation and management system.

## Monitoring and Logging

- **Prometheus**: [prometheus/prometheus](https://github.com/prometheus/prometheus)
  - Monitoring system and time series database.
- **Grafana**: [grafana/grafana](https://github.com/grafana/grafana)
  - Open-source platform for monitoring and observability.
- **Graylog**: [Graylog2/graylog2-server](https://github.com/Graylog2/graylog2-server)
  - Log management and analysis tool.
- **Nagios**: [nagios-plugins/nagios-plugins](https://github.com/nagios-plugins/nagios-plugins)
  - Network and server monitoring tool.
- **Zabbix**: [zabbix/zabbix](https://github.com/zabbix/zabbix)
  - Enterprise-class monitoring solution for networks and applications.
- **ELK Stack**:
  - **Elasticsearch**: [elastic/elasticsearch](https://github.com/elastic/elasticsearch)
  - **Logstash**: [elastic/logstash](https://github.com/elastic/logstash)
  - **Kibana**: [elastic/kibana](https://github.com/elastic/kibana)

## Backup and Recovery

- **Bacula**: [bacula-community/bacula](https://sourceforge.net/projects/bacula/)
  - A set of programs to manage backup, recovery, and verification of data across a network of computers.
- **Restic**: [restic/restic](https://github.com/restic/restic)
  - Fast, secure, and efficient backup program.
- **Duplicati**: [duplicati/duplicati](https://github.com/duplicati/duplicati)
  - Backup software to store encrypted, incremental backups on cloud storage services.
- **Amanda**: [zmanda/amanda](https://github.com/zmanda/amanda)
  - Advanced Maryland Automatic Network Disk Archiver, an open-source backup solution.

## Web Servers

- **Apache HTTP Server**: [apache/httpd](https://github.com/apache/httpd)
  - Open-source HTTP server for modern operating systems including UNIX and Windows.
- **Nginx**: [nginx/nginx](https://github.com/nginx/nginx)
  - High performance load balancer, web server, and reverse proxy.
- **Caddy**: [caddyserver/caddy](https://github.com/caddyserver/caddy)
  - Fast, multi-platform web server with automatic HTTPS.

## Virtualization and Containerization

- **Docker**: [docker/docker-ce](https://github.com/docker/docker-ce)
  - Platform for developing, shipping, and running applications in containers.
- **KVM (Kernel-based Virtual Machine)**: [qemu/qemu](https://github.com/qemu/qemu)
  - Open-source virtualization technology built into the Linux kernel.
- **VirtualBox**: [oracle/virtualbox](https://www.virtualbox.org/)
  - Powerful x86 and AMD64/Intel64 virtualization product.
- **Vagrant**: [hashicorp/vagrant](https://github.com/hashicorp/vagrant)
  - Tool for building and managing virtualized development environments.
- **Podman**: [containers/podman](https://github.com/containers/podman)
  - Tool for managing OCI containers and pods.

## Database Management

- **MySQL**: [mysql/mysql-server](https://github.com/mysql/mysql-server)
  - Open-source relational database management system.
- **PostgreSQL**: [postgres/postgres](https://github.com/postgres/postgres)
  - Advanced open-source relational database system.
- **MariaDB**: [MariaDB/server](https://github.com/MariaDB/server)
  - Open-source relational database management system forked from MySQL.
- **SQLite**: [sqlite/sqlite](https://www.sqlite.org/src)
  - C library that provides a lightweight, disk-based database.

## Terminal Multiplexers

- **tmux**: [tmux/tmux](https://github.com/tmux/tmux)
  - Terminal multiplexer that allows you to switch between several programs in one terminal.
- **screen**: [GNU Screen](https://www.gnu.org/software/screen/)
  - Terminal multiplexer that allows a user to access multiple separate terminal sessions inside a single terminal window or remote terminal session.

## Network File Systems

- **Samba**: [samba-team/samba](https://github.com/samba-team/samba)
  - SMB/CIFS file, print, and authentication server for UNIX.
- **NFS (Network File System)**: [nfs-utils](https://github.com/nfs-utils/nfs-utils)
  - Distributed file system protocol originally developed by Sun Microsystems.

## Security and Compliance

- **OpenSCAP**: [OpenSCAP/openscap](https://github.com/OpenSCAP/openscap)
  - Set of open-source libraries providing an easier path for making security and compliance easier.
- **Lynis**: [CISOfy/lynis](https://github.com/CISOfy/lynis)
  - Security auditing tool for UNIX-based systems.
- **OSSEC**: [ossec/ossec-hids](https://github.com/ossec/ossec-hids)
  - Open-source host-based intrusion detection system (HIDS).
